# CUDA & OpenCV with CMake 3.8

```
On Linux
	cd cuda-opencv-multithreading-with-cmake
	mkdir build
	cd build
	cmake ../ && make && ./test
On Windows
	cd cuda-opencv-multithreading-with-cmake
	mkdir build
	cd build
	cmake ../ -G"Visual Studio 12 2013" 
	# This creates a Visual Studio project in build directory which you can open in MSVS and run
```
